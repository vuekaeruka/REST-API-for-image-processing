Этот проект представляет собой REST API на языке Go для обработки изображений с возможностью загрузки, изменения размера, конвертации форматов и обрезки. Все операции могут выполняться как через веб-интерфейс, так и через HTTP-запросы.

Описание функционала

    Загрузка изображений — сохранение оригинальных файлов
    Изменение размера — изменение ширины и высоты изображения
    Конвертация форматов — преобразование между JPEG и PNG
    Обрезка изображений — вырезание фрагмента по указанным координатам
    Веб-интерфейс — удобная форма для всех операций

Веб-интерфейс
После запуска сервера откройте в браузере http://localhost:8080

Интерфейс предоставляет формы для:

    Загрузки изображений — файлы сохраняются в папку uploads
    Изменения размера — результат сохраняется в resized_images
    Конвертации форматов — результат сохраняется в converted_images
    Обрезки изображений — результат сохраняется в cropped_images
